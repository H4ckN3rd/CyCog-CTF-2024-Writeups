# Morse Code is Cool (50)

``
.. -....- --... ----- .---- -.. -....- -.-- ----- ..- -....- -- ----- .-. ... ...-- -....- -.-. ----- -.. . -....- .. ... -....- -.-. ----- ----- .----
``


Note : Wrap the answer in flag format. Flag Format: CCTF{}

## Solution

Simply copy and paste the morse code in online morse code decoder and wrap it in flag.

## Flag
Flag:-   CCTF{I-701D-Y0U-M0RS3-C0DE-IS-C001}
